# 模拟索引式文件系统  PyQt5 图形界面

一个课程作业级别的**文件系统模拟器**，实现了基本的索引式文件系统核心功能，并提供直观的 PyQt5 图形化界面操作。

#运行程序

- pip install PyQt5
- python main.py

## 功能概览

### 文件系统核心功能（IndexedFileSystem 类）

- 支持创建/删除文件与目录（支持多级目录）
- 支持读/写文件（覆盖写 + 部分追加场景）
- 支持文件复制、移动、重命名
- 支持目录复制（递归）、删除（递归）
- 简单的块分配与回收机制（固定块大小）
- 支持读写权限标记（rw / r / w / a）
- 文件系统状态持久化（pickle 序列化保存/加载）

### 图形界面功能（PyQt5）

- 左侧目录树（快速访问 + 展开/折叠）
- 右侧图标式文件视图（类似 Windows 资源管理器图标模式）
- 支持新建文件/文件夹
- 支持复制/粘贴/删除/重命名
- 支持前进/后退/向上导航
- 支持右键菜单、快捷键（Delete / Ctrl+C / Ctrl+V）
- 文件内容查看与简单编辑（文本文件）
- 属性查看（名称、大小、创建/修改时间、权限等）
- 一键格式化（重置文件系统）
- 关闭窗口自动保存文件系统状态

## 技术栈

- **核心语言**：Python 3.8+
- **GUI 框架**：PyQt5
- **持久化**：pickle（简单对象序列化）
- **开发环境**：Windows / PyCharm（推荐）
